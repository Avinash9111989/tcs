Provided projects are
1. InfyGoJSP - Registration module using JSP and Servlets
2. InfyGoMVC - Airline reservation application using Spring MVC and REST
3.AirScanner - A Rest client application to access the Flight API rest service
4. InfyGoMvcAngular - InfyGoMVC application with Angular as the front end

Both the maven and Non-Maven version are given.In case you are using the non-maven version, please add the required jars and its dependencies manually by referring the POM.xml of the Maven projects.
